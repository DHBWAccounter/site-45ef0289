# Bitcoin Association Switzerland

A Next.js clone of the Bitcoin Association Switzerland website.

## Getting Started

First, install dependencies:

```bash
pnpm install
```

Then, run the development server:

```bash
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Build

To build for production:

```bash
pnpm build
```

## Technologies Used

- Next.js 15.5.9
- React 19
- TypeScript
- Tailwind CSS
- Lucide React
